# Assets Folder

This folder should contain the following image files for the app:

## Required Files

1. **icon.png** - App icon (1024x1024px)
2. **adaptive-icon.png** - Android adaptive icon foreground (1024x1024px)
3. **splash.png** - Splash screen image (1242x2436px or similar)
4. **favicon.png** - Web favicon (48x48px)

## Icon Design Suggestions

### icon.png
- A cute cartoon doll face or full body
- Pink/gradient background matching the app theme
- Simple, recognizable design

### splash.png
- The same doll character in the center
- Pink gradient background (#FF69B4 to #FFB6C1)
- App name "AI Doll" or character name below

### adaptive-icon.png (Android)
- Transparent background with the doll character
- Will be overlaid on the background color defined in app.json

## Placeholder

Until you add real images, the app will use default Expo placeholders.
To generate proper icons, you can use:
- [Icon Kitchen](https://icon.kitchen/)
- [App Icon Generator](https://appicon.co/)
- Or create custom designs in Figma/Photoshop
